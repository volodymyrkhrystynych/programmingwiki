
## Plugins
:PlugStatus
- this lets you check the plugins installed when using vim-Plug as the plugin manager

## Comments
- Vim-commentary
- gcc to comment out line
- gc to comment out selected code

## Surround
- vim-surround helps surround text with quantifiers
- ys{motion}{char} to add
- dc{char} to delete
- cs{target}{replacement} to change

## Spell check
Spell check :help Spell
z-g : add to good list
z-w : add to bad list

]s : next misspelled word
[s : previous misspelled word
